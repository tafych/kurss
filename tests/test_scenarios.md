# Тестовые сценарии для складского приложения

## Сценарий 1: Регистрация и авторизация
1. Перейти на главную страницу
2. Нажать "Регистрация"
3. Заполнить форму:
   - Имя пользователя: test_user_01
   - Email: test01@example.com
   - Пароль: TestPass123
4. Нажать "Зарегистрироваться"
5. Проверить сообщение об успешной регистрации
6. Войти с созданными учетными данными
7. Проверить успешный вход

## Сценарий 2: Поиск товаров
1. Войти как пользователь
2. Перейти на страницу поиска
3. Ввести "ноутбук" в поле поиска
4. Выбрать категорию "Электроника"
5. Выбрать сортировку "По цене (убывание)"
6. Нажать "Найти"
7. Проверить результаты поиска
8. Нажать на первый товар в списке
9. Проверить детальную страницу товара

## Сценарий 3: Управление товарами (Админ)
1. Войти как администратор
2. Перейти в админ-панель
3. Нажать "Добавить товар"
4. Заполнить форму:
   - Название: Новый тестовый товар
   - Описание: Описание тестового товара
   - Артикул: TEST-NEW-001
   - Количество: 25
   - Цена: 9999.99
   - Категория: Электроника
5. Нажать "Добавить товар"
6. Проверить появление товара в списке
7. Нажать "Редактировать" у созданного товара
8. Изменить количество на 30
9. Сохранить изменения
10. Удалить тестовый товар

## Сценарий 4: Обработка ошибок
1. Попробовать войти с неверными данными
2. Попробовать зарегистрироваться с существующим email
3. Попробовать перейти на несуществующую страницу
4. Попробовать добавить товар с отрицательной ценой
5. Попробовать добавить товар с существующим артикулом

## Сценарий 5: API тестирование
1. Войти как пользователь
2. Перейти по адресу /api/products
3. Проверить JSON ответ
4. Перейти по адресу /api/products/1
5. Проверить детальную информацию о товаре
6. Попробовать получить несуществующий товар
